ChessIT — Xadrez 3D com Irrlicht, IrrAI e IA Neural

ChessIT é um projeto open source de um jogo de xadrez 3D desenvolvido em C++ utilizando a engine gráfica Irrlicht Engine, a biblioteca IrrAI para inteligência artificial e navegação, e SoLoud para o sistema sonoro.

O objetivo do projeto é criar uma experiência completa de xadrez 3D com:

- tabuleiro e peças renderizados em 3D;
- efeitos visuais com billboards e partículas;
- sistema de áudio e músicas clássicas;
- inteligência artificial tradicional (Minimax + Alpha-Beta);
- integração futura com modelos neurais ONNX treináveis;
- interface gráfica integrada à Irrlicht;
- replay de jogadas;
- níveis de dificuldade;
- arquitetura modular em C++17;
- Tecnologias Utilizadas;
- C++17;
- Irrlicht 1.7;
- IrrAI;
- SoLoud;
- ONNX Runtime (planejado);
- OpenGL / DirectX (via Irrlicht);
- Recursos Planejados;
- Sistema completo de regras do xadrez;
- IA do computador com múltiplos níveis;
- Aprendizado baseado em vetores treinados;
- Modelos neurais ONNX para análise de jogadas;
- Sistema de hover com efeitos visuais;
- Sons e trilha sonora configuráveis;
- Histórico e retorno de jogadas;
- Interface de configuração em tempo real;
- Compatibilidade Windows e Linux;
- Estrutura do Projeto;
- /media → modelos 3D, texturas, sons e cenários;
- /includes → headers das bibliotecas;
- /lib → bibliotecas estáticas;
- /bin → executáveis e DLLs;
- CHESSIT_POSITIONS.txt → posições do tabuleiro e peças;
- Objetivo do Projeto;

O projeto busca unir:

- programação gráfica 3D;
- IA para jogos;
- engines clássicas em C++;
- arquitetura modular;
- aprendizado de máquina aplicado ao xadrez;
